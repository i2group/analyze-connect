    Changelog for i2 Analyze REST SPI 1.0

    Version 1.0 is the first public release of the i2 Analyze REST SPI. As such, all endpoints and schemas are new at this version of the SPI.

    Added

    The following endpoints are new and fully supported at this version of the i2 Analyze REST SPI:

    • Endpoints for retrieving the configuration information from each connector
    • Endpoints for running synchronous queries
    • Endpoints for running asynchronous queries
    • An endpoint to authenticate users with third-party services
    • An endpoint for validating a request
