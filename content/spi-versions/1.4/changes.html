    Changelog for i2 Analyze REST SPI 1.4

    Version 1.4 contains minor changes to the i2 Analyze REST SPI.

    In this version, the "sensitive" headers that can be sent to a connector now include the i2 Analyze user identifier (I2-User-Id) and also that user's OIDC access token (I2-Oidc-Access-Token), if OIDC is the authentication mechanism for the i2 Connect gateway.
